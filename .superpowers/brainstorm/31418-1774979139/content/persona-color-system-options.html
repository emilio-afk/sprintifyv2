Sistema De Color Para "Por Persona"

Buscamos un sistema híbrido: el lane comunica riesgo/capacidad y los elementos internos comunican estado del trabajo. Te recomiendo la opción B.

A

Lane Casi Neutro

El lane usa color muy suave y la semántica vive sobre todo en chips y números. Más limpio, menos expresivo.

B

Híbrido Balanceado Recomendado

El lane tiene 4 tonos visibles pero sobrios. El interior usa otra escala para done / live / todo. Marca separada de semántica.

C

Semáforo Fuerte

Colores más saturados en todo el lane. Muy claro, pero puede sentirse ruidoso y competir con los datos.

Vista De La Opción Recomendada

Lane = riesgo/capacidad. Barra y pills = estado del trabajo. Verde sale de la semántica del lane y se reserva para UI global.
